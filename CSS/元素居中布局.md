# 元素居中布局

## 行内元素水平居中

如果目标元素内部是文本、`<span>`、`<a>` 或 `<img>` 等**行内元素**，通过设置其**父元素**的样式来实现水平居中。

```css
/* HTML: <div>Hello World</div> */
.parent-container {
    text-align: center; /* 使内部的文本和行内元素水平居中 */
}
```

## 块级元素水平居中

如果目标元素是 `<div>`、`<p>` 等**块级元素**，且它设置了固定的宽度（`width`）。

```css
/* HTML: <div class="box">Box</div> */
.box {
    width: 200px; /* 必须设置宽度才能使用 margin: auto */
    margin: 0 auto; /* 上下边距为 0，左右边距自动分配 */
}
```

## flex水平垂直居中

设置父元素

```css
/* HTML: <div class="flex-container"><div>Centered Item</div></div> */
.flex-container {
    display: flex;
    height: 400px; /* 必须设置高度才能进行垂直居中 */
    
    /* 核心居中属性 */
    justify-content: center; /* 主轴 (水平) 居中 */
    align-items: center;     /* 交叉轴 (垂直) 居中 */
}
```

在 Flexbox 容器内，子元素也可以使用 `margin: auto;` 来实现自动对齐和居中。

```css
.flex-container {
    display: flex;
    height: 400px;
}
.centered-item {
    margin: auto; /* 在 flex 容器内，自动填满所有方向的剩余空间 */
}
```

## grid水平垂直居中

```css
/* HTML: <div class="grid-container"><div>Centered Item</div></div> */
.grid-container {
    display: grid;
    height: 400px;
    
    /* place-items 是 align-items 和 justify-items 的缩写 */
    place-items: center; 
}
```

## 绝对定位居中

```css
/* HTML: <div class="relative-parent"><div class="absolute-center">Center</div></div> */
.relative-parent {
    position: relative;
    height: 400px;
}
.absolute-center {
    position: absolute;
    top: 50%;         /* 移动元素左上角到中心点 */
    left: 50%;
    transform: translate(-50%, -50%); /* 依靠自身尺寸，回退 50% 宽度和高度 */
}
```
